# Web Development Project Estimator in Laravel with Vue.JS

The Web Development Project Estimator is a simple tool that allows web designers and site developers to quickly and thoroughly estimate the time and materials required for a proposed web project.

To use, simply enter the title of the project and your default hourly rate. Then adjust your anticipated hours accordingly to generate your total project estimate. When finished, you can view your finalized estimate in a print-ready format in case youâ€™d like to save a PDF or print a copy for your records.

Run Below commands

For creating Laravel & Auth Scaffolding
```sh
composer create-project laravel/laravel estimator
composer require laravel/ui
php artisan ui vue --auth
npm install
npm run dev
php artisan migrate
php artisan serve
```
